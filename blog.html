Important FAQs

What is the Document Object Model (DOM)?

The Document Object Model (DOM) connects web pages to scripts or programming languages by representing the structure of a document. The DOM represents a document as a tree structure. Each branch of the tree ends in a node, and each node contains objects.


With the Document Object Model, programmers can create and build documents, navigate their structure, and add, modify, or delete elements and content. Anything found in an HTML document can be accessed, changed, deleted, or added using the Document Object Model (DOM) methods.

How do you select an element from the DOM?

You can select an element by bringing the ID name, Class name, or by using query selector in Javascript. For example: to call a specific element you give an ID name and create a variable and use -
const anyName = document.getElementById()
And in the brackets should be the unique name of the ID you have given within the double or single quotation marks.


And if it is only for one particular class then use -
const anyName = document.getElementByClassName()


You can also get an element by its tag name using this method - document.getElementByTagName()


As for query selector if it is for a class then you do -
const anyName = document.querySelectorAll('.className') - here you use the class identifier sign.


And if it is for a query selector id then you use -
const anyName = document.querySelector('#idName') - here you use id identifier sign with the name.


There are many different ways of selecting an element from the DOM, these are just a few from the many.

What is event delegation in the context of the DOM , and why is it useful?

Event delegation in the context of the DOM is a Javascript technique that uses propagation to handle events at a higher level in the DOM tree.


It is useful because it attaches a single event listener to a parent element instead of attaching it to each child element, it thereby improves the performance, simplifies event handling and supports dynamic elements.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attribute using the DOM which is - document.getElementById() and document.querySelector() however these are methods used for single elements, and the document.querySelector() method is more robust as it can select an element on the page by any type of selector.


You can also use the querySelectorAll() method to grab all elements with the same class name and use forEach() to loop through them and apply changes to it. And it is also possible to access a specific element with querySelectorAll() the same way you would with an array by using bracket notation.

One of the ways for manipulating an element's styles using the DOM, is by using the setAttribute() method, however this will replace all the existing inline styles from the element. That's why it is better to use the style attribute directly.
For example: const a = document.querySelector('div');
a.style.border = '2px solid blue';
a.style.backgroundColor = 'gray';


For better understanding different colors have been applied for this DOM manipulation example. Firstly, you set the style attribute (red color), followed by a dot, then you set the property name (purple color), and then again followed by a dot you then set the value (green color) within the single quotations.
Overall, attributes are values that contain additional information about HTML elements. They usually come in name/value pairs, may be essential depending on the element.
